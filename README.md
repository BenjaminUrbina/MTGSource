# MTGSource

## Descripción del proyecto

**MTGSource** es un proyecto en fase de desarrollo cuyo objetivo es crear una aplicación **web y móvil** para la gestión centralizada de cartas de **Magic: The Gathering (MTG)**.

La aplicación permitirá:
- Escanear cartas utilizando la cámara del teléfono móvil.
- Almacenar las cartas en una base de datos.
- Visualizar, organizar y compartir el stock de cartas en tiempo real.

La idea surge a partir de un problema común entre jugadores, coleccionistas y tiendas: a medida que la colección crece, se vuelve difícil llevar un control claro de **qué cartas se tienen y en qué cantidad**.

Actualmente existen herramientas como Moxfield o ManaBox que permiten escanear y exportar cartas (por ejemplo a CSV), pero estas soluciones están **fragmentadas** y no cubren de forma integral necesidades como inventario centralizado, visualización en tiempo real y uso comercial.

MTGSource busca **unificar estas funcionalidades en una sola plataforma**, pensada tanto para:
- **Coleccionistas**, que desean ordenar y visualizar su colección.
- **Empresas o tiendas**, que necesitan gestionar stock de forma simple, clara y actualizada.

---

## 🎯 Objetivos principales

- Centralizar el inventario de cartas MTG en una única aplicación.
- Facilitar el escaneo y registro de cartas desde dispositivos móviles.
- Permitir la visualización del stock en tiempo real.
- Sentar las bases para futuras funcionalidades orientadas a uso comercial.
